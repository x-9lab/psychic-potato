# 精神小土豆

使用 vite 与 antd 的 react 种子项目

已上传了 `yarn.lock`，为了防止依赖版本不一致，开发过程使用 `yarn`

## 开发配置
- 需要 nodeJs 环境，推荐 v12 或以上版本
- 安装依赖
    ```shell
    yarn
    ```
- 本项目基于 [Create React App](https://github.com/facebook/create-react-app)
    ```shell
    # 编译开发环境
    yarn dev

    # 编译生产环境
    yarn build
    ```
